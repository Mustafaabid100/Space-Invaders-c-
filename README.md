# Space-Invaders-c-
A 2D Space Invaders–style game developed in C++ using the Windows API. The project features
real-time gameplay with player movement, shooting mechanics, and collision detection. It includes
multiple difficulty levels, a menu system, and game state management. A file-based persistence system
enables save/load functionality and high-score tracking.
Features
1 Real-time player movement and shooting
2 Enemy movement with difficulty levels
3 Collision detection and scoring system
4 Pause, save, and load functionality
5 High-score tracking using file I/O
Tech Stack
1 C++
2 Windows API (windows.h, mmsystem.h)
3 File handling
Setup Instructions
1 Ensure Windows OS and a C++ compiler (MinGW or Visual Studio)
2 Compile using: g++ main.cpp -o game -lwinmm
3 Run using: game.exe
Controls
1 Arrow Keys: Move player
2 Shift: Shoot
3 P / ESC: Pause
4 S: Save game
5 L: Load game
6 M: Return to menu
